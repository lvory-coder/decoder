# decoder
Универсальный декодер
================================
DECODER
================================

DESCRIPTION:
Universal decoder for CTF challenges. Supports multiple ciphers
and encodings with automatic detection and flag search.

FEATURES:
- 16+ cipher/encoding types
- Auto-detection mode
- CTF flag search
- XOR with key input
- Clipboard support
- File loading/saving
- Colorful console interface

HOW TO USE:
1. Double-click RUN_ME.bat or CTFDecoder.exe
2. Follow on-screen instructions
3. Select cipher type or use auto-detect
4. Enter your encoded text
5. View results and find flags!

SUPPORTED FORMATS:
- Base64, Base32, Base16
- Hex, Binary
- Caesar, ROT13, ROT47, Atbash
- XOR (with key or bruteforce)
- URL encoding
- Morse code (basic)
- Rail Fence
- Reverse

REQUIREMENTS:
- Windows 7/8/10/11
- Python 3.8+ (for .py version)
- 10MB free space
